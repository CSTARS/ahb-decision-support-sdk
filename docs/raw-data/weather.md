Weather data for the poplar 3pg growth model.  For more information please see the 
[poplar-3pg-model](https://github.com/CSTARS/poplar-3pg-model)

## Structure

```js
{
  "1" : {
      "month": Number,
      "tmin": Number,
      "tmax": Number,
      "tdmean": Number,
      "ppt": Number,
      "rad": Number,
      "daylight": Number,
      "nrel": Number,
  },
  ...
}
```
